# Bomberman

A multiplayer Bomberman game built with Node.js and Socket.io.

**Built by [Suchir Balaji](https://github.com/suchir)**

## How to Play

1. Enter a username
2. Create a game or join an existing lobby
3. Use arrow keys to move, spacebar to place bombs
4. Be the last player standing!

## Running Locally

```bash
npm install
npm start
```

Then open http://localhost:4242 in your browser.

## Deployment

This game can be deployed to platforms like Railway, Render, or Fly.io that support Node.js and WebSockets.
